# Codexa Features

Codexa is a powerful AI-powered code editor designed for modern developers. Here's a complete overview of all features.

## Core Editing Features

### Monaco Code Editor
- **Syntax Highlighting** - Support for 50+ programming languages (JavaScript, TypeScript, Python, Java, C++, Go, Rust, SQL, etc.)
- **IntelliSense** - Intelligent code completion and suggestions
- **Minimap** - Visual overview of your entire code file
- **Line Numbers** - Easy navigation through code
- **Bracket Matching** - Automatic bracket and parenthesis highlighting
- **Multi-Cursor Support** - Edit multiple lines simultaneously
- **Code Folding** - Collapse and expand code sections
- **Search & Replace** - Find and replace with regex support

### File Management
- **File Explorer** - Visual tree view of all your files
- **Create Files** - Quickly create new code files
- **Rename Files** - Edit file names with one click
- **Delete Files** - Remove files from your project
- **File Persistence** - All files saved automatically in browser storage
- **Multiple Tabs** - Work on multiple files at once
- **File Type Detection** - Automatic language detection

## AI-Powered Features

### AI Code Assistant
- **Real-time Chat** - Conversational AI assistant integrated in the editor
- **Code Generation** - Ask AI to write code from natural language descriptions
- **Code Explanation** - Get detailed explanations of complex code
- **Code Refactoring** - Improve code quality and performance
- **Bug Fixing** - Automatically identify and fix bugs in your code
- **Code Review** - Get suggestions for code improvements

### AI Models

- **Context-Aware** - AI understands your code context
- **Real-time Responses** - Stream responses as they're generated
